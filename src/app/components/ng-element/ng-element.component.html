Learn Angular

Directives

Directives in Angular are used to extend the functionality of existing elements.

eg: *ngIf eg: *ngFor eg: *ngSwitch
Pipes

Pipes in Angular are used to transform data before displaying it in the template.

eg: |uppercase eg: customPipe
Services

Services in Angular are used to share data and functionality across components.

eg: DataService eg: AuthService eg: NotificationService
 Dependency Injection

Dependency Injection in Angular is a design pattern used to manage dependencies between components and services.

eg: constructor(private dataService: DataService) Routing

Routing in Angular is used to navigate between different views in the application.

eg: routerLink eg: RouterModule.forRoot(routes)
